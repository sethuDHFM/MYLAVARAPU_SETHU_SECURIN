# Recipe Assessment Project

## Setup Instructions

### Backend Setup
1. cd backend
2. pip install -r requirements.txt
3. Set up PostgreSQL database
4. Run: python scripts/parse_json.py
5. Run: python app.py

### Frontend Setup
1. cd frontend
2. npm install
3. npm start

### Database Setup
1. Create PostgreSQL database: recipes_db
2. Run schema.sql
